Install rust (latest stable): https://www.rust-lang.org/tools/install  

Get test rom from https://github.com/retrio/gb-test-roms/raw/master/cpu_instrs/cpu_instrs.gb

Building:

```shell
cargo build
```

Running:  
```shell
cargo run cpu_instrs.gb
```

Keybindings:
- esc -> toggle ui
- wasd -> direction pad
- j,k -> b,a
- u,i -> select, start
- shift -> turbo
